**LPTV Dashboard**

Веб-приложение для Дизайн студии интерьеров с учётками и админ панелью, цель которой вести ежедневный учёт рабочего времени по проектам, с базовой аналитикой по проектам, дашбордом, интерактивными таблицами и прочее.
